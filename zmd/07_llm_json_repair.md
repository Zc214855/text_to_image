# 阶段 7：智谱 JSON 解析修复

## 问题

智谱成功完成了故事分析，但在 `story_text` 字符串中输出了未转义的 ASCII 双引号：

```text
"story_text": "阿圆问奶奶："月亮是不是迷路了？"奶奶回答。"
```

该内容不是合法 JSON，因此程序在进入 Kolors 图片生成前终止。Kolors 未收到任何请求。

## 修复

1. 标准 `json.loads()` 仍作为首选解析路径。
2. 标准解析失败时使用 `json-repair` 修复常见的引号、逗号和截断错误。
3. 系统提示词要求中文对白使用 `「」`，禁止在 JSON 字符串中直接使用未转义 ASCII 双引号。
4. 修复后仍校验根节点必须为 JSON 对象。
5. 新增未转义中文对白的回归测试。
