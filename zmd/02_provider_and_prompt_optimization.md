# 阶段 2：供应商接入与提示词优化

## 已完成

1. 接入智谱 GLM-5.1 的腾讯云 Token Plan OpenAI 兼容地址。
2. 接入火山方舟 LLM 配置。
3. 接入火山方舟 Seedream 5.0 Lite 图片生成接口。
4. 新增火山推荐的 2K 横竖屏尺寸，关闭图片水印。
5. 将真实密钥迁移到 Git 忽略的 `.env.local`，示例文件改为占位符。
6. 分镜方式从“每 30-50 字一张”改为“一个完整可见动作一个镜头”，约每 1.5 个句子合并为一镜，最多 18 镜。
7. 要求所有可见角色先定义固定外观，包括只出现一次的角色。
8. 将动作、环境、构图、景别、光线拆为结构化字段。
9. 由代码按固定顺序组装最终提示词，消除角色和风格重复。
10. 为 SiliconFlow 和 DashScope 增加反向提示词。
11. 为所有模型增加更严格的无文字、无标识、无水印约束。

## 质量提升边界

- 纯文生图只能提高“描述一致性”，不能保证跨图片像素级角色一致。
- 更强的一致性需要角色参考图、图生图或火山方舟组图能力。
- 当前版本优先修复错误提示词和模型接入，未引入自动参考图链路，避免改变现有生成流程和费用模型。
